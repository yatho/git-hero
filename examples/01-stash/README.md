# 08 - Git Stash
> Save work temporarily without commits
**Time:** 3-4 minutes

## Quick Start
```bash
./setup.sh && ./demo.sh
```

## Key Commands
```bash
git stash push -m "message"  # Save work
git stash list               # View stashes
git stash pop                # Restore and remove
git stash apply              # Restore but keep in stash
```
